## Run Project

Clone Project

```git
git clone https://github.com/iury0393/proffy_nlw2
```

API

```ssh
cd api
npm install
npm run knex:migrate
npm run knex:seed
npm run dev
```

Web

```ssh
cd web
npm install
npm run start
```

Mobile

```ssh
cd mobile
npm install
npm run start
```
